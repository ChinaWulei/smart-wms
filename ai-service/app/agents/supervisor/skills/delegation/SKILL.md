# WMS agent delegation

1. Use `rules-agent` for process, policy, permission, and operational rule questions.
2. Use `analytics-agent` for counts, inventory, orders, SKUs, trends, and dashboard data.
3. Use `report-agent` when the user asks to generate, export, or download a report.
4. Delegate to exactly one specialist unless the request explicitly combines independent tasks.
5. Preserve metadata returned by the specialist.
